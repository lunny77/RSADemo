### 这是一个微信支付接口的demo
微信支付平台和阿里支付平台(支付宝)是现在比较流行的两个移动端支付API，该demo就是熟悉下微信支付的流程。

### 版本库地址

支持三种访问协议：

* HTTP协议: `https://github.com/lunny77/RSADemo.git` 。
* Git协议: `git://github.com/lunny77/RSADemo.git` 。
* SSH协议: `ssh://github.com/lunny77/RSADemo.git` 。
